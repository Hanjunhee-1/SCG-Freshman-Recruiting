/ => 여기서는 /stacks 로 redirect() 를 해주어야 함. (안한다면 감점)

GET /stacks
    메인 페이지. 
    검색바 제공(제목 검색, ID번호 검색)
    현재 게시글 개수와 게시글 목록 제공
    '새 게시글 작성하기' 라는 버튼 제공
    제목을 클릭했을 때 해당 게시글 상세보기로 이동가능해야함.


GET /stacks?title=_title
    제목을 검색하여 검색된 결과물들을 표시하는 페이지
    검색방식은 저장된 게시글들의 제목 중에서 해당 키워드를 포함하는 게시물들을 모두 검색 가능해야 함.

POST /stacks/:id
    작성할 글의 ID 번호 검색을 통해 들어올 수 있는 페이지
    게시글의 제목과 내용, 작성일, 수정일이 표시됨. 

GET /stacks/write
    게시글을 작성하는 폼 제공
    ++ 폼을 꾸민다면 가산점(+5)

POST /stacks/:id/edit
    게시글을 수정하는 곳.
    placeholder 를 통해 기존에 저장되어 있는 값을 보여주기
